Dark Mode for Minecraft Bedrock Edition.

Get Dark Mode in Minecraft Bedrock Edition. Just download the pack and activate in Global Resources, World Resource Packs or Realms.

Features

Makes every UI darker

Protects your eyes while playing Minecraft

Issues

Due to Mojang changing every UI from Json UI to Ore UI which is hardcoded and to edit requires a third-party launcher, some UIs could not be darkened.
